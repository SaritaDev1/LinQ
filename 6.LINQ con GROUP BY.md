# LINQ con GROUP BY

`GROUP BY` se utiliza para agrupar datos que tienen algo en común.

Esto significa que no se trabaja con cada registro por separado, sino que se forman grupos según un campo específico.

Por ejemplo, se puede agrupar clientes por estado, préstamos por cliente, empleados por cargo o facturas por cliente.

Ejemplo: contar clientes por estado

```csharp
public object ContarClientesPorEstado()
{
    var consulta = from c in db.Clientes
                   group c by c.Estado into grupo
                   select new
                   {
                       Estado = grupo.Key,
                       Cantidad = grupo.Count()
                   };

    return consulta.ToList();
}
