# Capa de acceso a datos

La capa de acceso a datos es la parte del sistema que se encarga de comunicarse con la base de datos.

LINQ participa principalmente en esta capa, porque aquí se realizan las consultas para obtener información.

LINQ funciona escribiendo consultas dentro de C# para traer información de la base de datos. En este manual se toma como referencia `Northwind`, donde existen tablas como `Customers`, `Products` y `Orders`.

Por ejemplo, si se necesita listar los clientes, LINQ permite consultar la tabla `Customers` desde el código:

```csharp
public List<Customer> ListarClientes()
{
    var consulta = from c in db.Customers
                   select c;

    return consulta.ToList();
}
```

En este ejemplo, `db.Customers` representa la tabla `Customers` de `Northwind`. La parte `select c` indica que se seleccionan los clientes.

Por eso, LINQ se usa en la capa de acceso a datos: porque ayuda a obtener información de la base de datos de forma ordenada antes de enviarla a la capa de negocio.
