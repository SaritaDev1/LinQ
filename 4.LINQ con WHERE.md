# LINQ con WHERE

WHERE se utiliza para filtrar datos mediante una condición.

Esto significa que no se muestran todos los registros, sino solamente los que cumplen una regla específica.

Por ejemplo, si se tiene una tabla de clientes y solo se desea mostrar los clientes activos, se puede usar where.

```csharp
public List<Cliente> ObtenerClientesActivos()
{
    var consulta = from c in db.Clientes
                   where c.Estado == "Activo"
                   select c;

    return consulta.ToList();
}
