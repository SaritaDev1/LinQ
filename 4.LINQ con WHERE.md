# LINQ con WHERE

`WHERE` se utiliza para filtrar datos mediante una condición.

Esto significa que no se muestran todos los registros, sino solamente los que cumplen una regla específica.

LINQ con `where` puede usarse en la capa de acceso a datos y también en la capa de negocio, pero en cada una se utiliza de diferente manera.

En la capa de acceso a datos, `where` se usa para consultar directamente la base de datos y traer solo los datos necesarios.

En la capa de negocio, `where` se usa para filtrar o revisar información que ya fue obtenida desde la capa de acceso a datos.
