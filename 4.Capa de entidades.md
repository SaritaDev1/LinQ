# Capa de entidades

La capa de entidades es la parte del sistema donde se crean las clases que representan las tablas de la base de datos.

Una entidad se puede entender como una representación en C# de una tabla. Cada columna o atributo de la tabla se convierte en una propiedad dentro de la clase.

En este manual se toma como ejemplo la base de datos `Northwind`, que contiene tablas como clientes, empleados, pedidos, productos y categorías.

Por ejemplo, la tabla `Customers` puede representarse con una clase llamada `Customer`.

```csharp
public class Customer
{
    public string CustomerID { get; set; }
    public string CompanyName { get; set; }
    public string ContactName { get; set; }
    public string City { get; set; }
    public string Country { get; set; }
}
```

En este ejemplo, la clase `Customer` representa la tabla `Customers`, y sus propiedades representan algunas columnas de esa tabla.

También se pueden crear otras entidades según las tablas que se necesiten usar:

```text
Customers     → Customer
Products      → Product
Orders        → Order
Employees     → Employee
Categories    → Category
Order Details → OrderDetail
```

La capa de entidades no es donde se escriben principalmente las consultas LINQ, pero sí participa de forma indirecta, porque LINQ trabaja usando estas clases y sus propiedades.

Por ejemplo, al consultar `Customers` con LINQ, cada registro puede representarse como un objeto de la clase `Customer`.

```csharp
var consulta = from c in db.Customers
               select c;
```

La capa de entidades permite representar las tablas de `Northwind` como clases de C#, para que LINQ pueda consultar y manejar esos datos de forma ordenada.
