# LINQ con ORDER BY

`ORDER BY` se utiliza para ordenar los datos antes de mostrarlos.

Esto significa que la información puede aparecer en un orden específico, por ejemplo, por nombre, apellido, fecha, valor o código.

Por ejemplo, si se tiene una tabla de clientes y se desea mostrar los clientes ordenados por nombre, se puede usar `orderby`.

```csharp
public List<Cliente> ListarClientesOrdenados()
{
    var consulta = from c in db.Clientes
                   orderby c.Nombre
                   select c;

    return consulta.ToList();
}
